# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, critique ideas and code — never
people. Harassment, personal attacks and bad-faith behavior are not tolerated.

Reports: vhanssen@menneske.no. Reports are handled confidentially.
